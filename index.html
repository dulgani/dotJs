Un besoin un object

Dans le cadre de mon travail j'ai eu récemment un petit soucie avec un client qui avait absolument besoin d'avoir un retour visuel sur les interactions dans son application mobile. Je me suis donc créé un petit objet JavaScript que j'ai tout simplement appelé Dot parce qu'il sert à faire des points (super l'originalité!!)

Presentation

Ce Dot permet d'avoir, à partir de n'importe quel événement que l'on souhaite, un retour visuel sous forme d'un cercle qui grandit (keyframe CSS avec du scale et de l'opacity) et qui finie par disparaitre. Cela permet d'avoir un retour visuel lors d'un click un touch, un keypress etc

Utilisation

l'Utilisation est très simple, il suffit d'importer le fichier JS dans son code HTML et d'initialiser le Dot avec ce dont on a besoin comme paramètres de cette façon

     init('non_event', {
            id: 'dot',
            className: 'dot',
            elem: 'div',
            position: 'absolute',
            background: '#FF0000',
            width: 20,
            height: 20,
            radius: 50,
            opacity: 0,
        });

et c'est tout, dès que l'event ciblé est capturé, l'animation se déclenche et lance le retour visuel.

Dans un avenir incertain

Je n'ai pas encore pensé à de futur amélioration de ce petit Dot mais je voulez le partager avec vous, si cela peut rendre service, comme cela me la fait à moi, je serais déjà content.
